    • JSON oznacza JavaScript Object Notation. JSON to format tekstowy do przechowywania i transportu danych. JSON jest samoopisujący się i łatwy do zrozumienia.


    JSON (cechy / własności)…

    • jest to lekki format wymiany danych,
    • jest to zwykły tekst napisany w notacji obiektowej JavaScript,
    • służy do przesyłania danych pomiędzy komputerami,
    • jest niezależny od języka programowania.

    Rozszerzenie pliku

    Plik zapisujemy z rozszerzeniem .json. Pliki możemy otworzyć za pomocą dowolnego edytora tekstowego.


    Zastosowanie

    Ponieważ format jest tylko tekstowy, dane JSON można łatwo przesyłać między komputerami i używać w dowolnym języku programowania.


    JSON - składnia

    • Dane reprezentowane są poprzez parę klucz:wartość,
    • Dane oddzielone są przecinkami
    • Nawiasy klamrowe reprezentują obiekty,
    • Nawiasy kwadratowe reprezentują tablice.

    Walidacja

    Na stronie jsonlint.com możesz zweryfikować poprawność pliku JSON (to bardzo przydatne narzędzie).


    Struktura JSONa


         "klucz": wartość
    1. klucz powinien być typu:
    • string.
    1. wartość może być typu:
    • string,
    • number,
    • object,
    • array,
    • boolean,
    • null.
