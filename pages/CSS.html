    Dokumentacja

    Dostęp do dokumentacji: https://developer.mozilla.org/en-US/docs/Web/CSS.


    Struktura css

        p {
        color: green;
        font-size: 20px;
    }

    Zwróć uwagę!!!

    • p to selektor (jest odpowiednikiem znacznika występującego w języku HTML np. <p>),
    • po selektorze występują nawiasy klamrowe { } - w nich zawarte są polecenia formatujące,
    • color, font-size to cecha (określana jest przez specyfikację),
    • green, red to wartość (definiowana jest przez programistę w zależności od potrzeb),
    • cechę i wartość rozdzielamy :,
    • po wartości powinien występować średnik ;.

    Gdy w dokumencie HTML wystąpi taki znacznik:

        <p>ANS Tarnów</p>

    Dla zaprezentowanego selektora efekt będzie następujący:

    Akapit zawiera czcionkę o rozmiarze 12px koloru zielonego (green).


    Typy selektorów


    Selektor elementu

    Kod CSS:

        p {
        color: green;
        font-size: 20px;
    }

    Kod HTML:

        <p>ANS Tarnów</p>

    Selektor identyfikatora

    Kod CSS:

        #my-id {
    color: green;
    font-size: 20px;
    }

    lub

        p#my-id {
    color: green;
    font-size: 20px;
    }

    Kod HTML:

        <p id="my-id">ANS Tarnów</p>

    Selektor klasy

    Kod CSS:

        .my-class {
    color: green;
    font-size: 20px;
    }

    lub

        p.my-class {
    color: green;
    font-size: 20px;
    }

    Kod HTML:

        <p class="my-class">ANS Tarnów</p>

    Selektor atrybutu

    Kod CSS:

        p[title="description"] {
    color: green;
    font-size: 20px;
    }

    Kod HTML:

        <p title="description">ANS Tarnów</p>

    Selektor pseudoklasy

    Kod CSS:

        a:hover {
    background-color: yellow;
    }

    Kod HTML:

        <a href="#">Destination</a>

    Lokalizacja styli

    Styl lokalny


    Plik HTML:

        <!DOCTYPE html>
    <html lang="pl">
        <head>
            <meta charset="utf-8">
            meta name="description" content="Opis">
            <meta name="keywords" content="Słowo kluczowe 1, Słowo kluczowe 2">
            <meta name="author" content="Tomasz Gądek">
            <meta name="viewport" content="width=device-width, initial-scale=1">
                    
            <title>css</title>
        </head>
        <body>
            <p style="font-size: 20px; color: red;">ANS Tarnów</p>
        </body>
</html>

    Styl zagieżdżony

    Plik HTML:

        <!DOCTYPE html>
    <html lang="pl">
        <head>
            <meta charset="utf-8">
            <meta name="description" content="Opis">
            <meta name="keywords" content="Słowo kluczowe 1, Słowo kluczowe 2">
            <meta name="author" content="Tomasz Gądek">
            <meta name="viewport" content="width=device-width, initial-scale=1">
                
            <title>css</title>
            <style type="text/css">
                p {
                    font-size: 20px;
                    color: red;
                }
            </style>
        </head>
        <body>
            <p>ANS Tarnów</p>
        </body>
    </html>

    Styl zewnętrzny

    Plik HTML:

        <!DOCTYPE html>
    <html lang="pl">
        <head>
            <meta charset="utf-8">
            <meta name="description" content="Opis">
            <meta name="keywords" content="Słowo kluczowe 1, Słowo kluczowe 2">
            <meta name="author" content="Tomasz Gądek">
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <link rel="stylesheet" href="style.css">
                
            <title>css</title>
        </head>
        <body>
            <p>ANS Tarnów</p>
        </body>
    </html>

    Plik CSS:

        p {
        font-size: 20px;
        color: red;
    }
