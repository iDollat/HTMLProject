    • JavaScript (JS) skryptowy język programowania stworzony przez firmę Netscape. Używany do tworzenia i kontrolowania dynamicznej zawartości strony internetowej bez konieczności jej przeładowania.

    JavaScript jest językiem interpretowanym. Kod jest wykonywany od góry do dołu, a wynik jest zwracany natychmiast.


    Rozszerzenie pliku

    Plik zapisujemy z rozszerzeniem .js. Pliki możemy otworzyć za pomocą dowolnego edytora tekstowego.


    Składania JavaScript (C vs JavaScript)

    Zmienne

    JavaScript to język luźnej kontroli typów.

    Zmienne - język C:

        int a = 7;
    char b = 'a';
    float c = 6.4;
    char *d = "abcd";

    Zmienne - język JavaScript:

        let a = 7;
    let b = 'a';
    let c = 6.4;
    let d = "abcd";

    Funkcje

    Funkcje - język C:

        int add(int a, int b) {
    return a + b;
    }

    Funkcje - język JavaScript:

        function add(a, b) {
    return a + b;
    }

    Instrukcje sterujące

    Instrukcje sterujące - język JavaScript:

        let age = 18;
                
    if(age >= 18) {
        console.log("Adult!");
    } else {
        console.log("Teen!");
    }

    Pętle programowe

    Pętle programowe - język JavaScript:

        for(let i = 0; i < 5; i++) {
    console.log(`counter = ${i}`);
    }

    Struktury

    Struktury - język C:

        struct User {
    char name;
    int age;
    };
                
    struct User user;
                
    user.name = 'a';
    user.age = 12;

    Struktury - język JavaScript:

        let user = {
    name: 'b',
    age: 1
    };
                
    user.name = 'a';
    user.age = 12;

    Zmienne tekstowe

    Zmienne tekstowe - język JavaScript:

        let text1 = "Veni Vidi Vici";
    let text2 = 'Veni Vidi Vici';

    Tablice

    Tablice - język JavaScript:

        let tab = [];
    tab.push("lorem ipsum");
    tab.push(123);
                
    console.log(tab);   // Array [ "lorem ipsum", 123 ]

    Praca z konsolą i typeof

        let a = 'text';
    let b = "text";
    let c = 1;
    let d = { name: "Tom" };
                
    console.log(typeof a); // string
    console.log(typeof b); // string
    console.log(typeof c); // number
    console.log(typeof d); // object

    Lokalizacja skryptów JavaScript

    Osadzony skrypt JavaScript:

        <!DOCTYPE html>
    <html lang="pl">
        <head>
            <meta charset="utf-8">
            <meta name="description" content="Opis">
            <meta name="keywords" content="Słowo kluczowe 1, Słowo kluczowe 2">
            <meta name="author" content="Tomasz Gądek">
            <meta name="viewport" content="width=device-width, initial-scale=1">
                
            <title>js</title>
                
            <!-- osadzony skrypt js -->
            <script type="text/javascript">
                console.log("Hello JS!");
            </script>
        </head>
        <body>
                
        </body>
    </html>

    Zewnętrzny skrypt JavaScript:

        <!DOCTYPE html>
    <html lang="pl">
        <head>
            <meta charset="utf-8">
            <meta name="description" content="Opis">
            <meta name="keywords" content="Słowo kluczowe 1, Słowo kluczowe 2">
            <meta name="author" content="Tomasz Gądek">
            <meta name="viewport" content="width=device-width, initial-scale=1">
                
            <title>js</title>
                
            <!-- zewnetrzny skrypt js -->
            <script type="text/javascript" src="./script.js"></script>
        </head>
        <body>
                
        </body>
    </html>
